# 学习计划

## 学习路径建议

基于学员画像（Web3 基础好，能借助 AI 开发），推荐以下学习路径：

### 阶段一：AI 基础补强（1-2 周）
- LLM、Prompt、Context 核心概念
- RAG 与外部知识接入
- Agent 与工具调用基础
- MCP 协议理解

### 阶段二：AI × Web3 Bridge（2-3 周）
- Chain-aware Context：链上状态进入 Agent 上下文
- Web3 Tool Use：RPC、钱包、合约工具调用
- Agent Workflow：自动化边界与 human-in-the-loop
- Agent Wallet：权限管理与 Session Key
- Machine Payment：机器间支付结算

### 阶段三：AI 安全与前沿（2-3 周）
- AI Security：Prompt Injection、权限隔离
- AI Privacy：隐私边界
- Verifiable AI：执行可验证
- Agent Trust & Reputation：声誉记录

### 阶段四：Hackathon 项目实战
- 从前沿探索中选择方向
- 组合知识节点开发小原型
- 准备项目展示与提交

## 每日学习节奏

- **早上**: 读取 WCB Learning 页面，确认当日任务
- **晚上**: 完成学习、记录打卡、整理实验
